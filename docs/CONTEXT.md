# Contexto do Projeto: HeroFracture

## Objetivo
HeroFracture é um jogo 2D de plataforma, ação e combate ("Runner Gun"), desenvolvido na Unity Engine versão 6000.4.1f1. O objetivo é proporcionar uma jogabilidade intensa baseada em reflexos, ataques corpo-a-corpo e evolução contínua.

## História (Lore)
A megacorporação Fracture Tech utiliza fendas temporais para sequestrar heróis de diversas eras, submetendo-os a uma lavagem cerebral para transformá-los em armas biônicas sob seu controle. O Samurai, nosso protagonista, conseguiu despertar durante o processo de condicionamento mental. Agora, ele precisa escapar desta instalação hostil, lutando contra seus ex-aliados controlados pela corporação, para encontrar o caminho de volta para casa.

## Pilares do Jogo
1. **Ação Rápida e Combate:** Foco em pular, dar dashes e atacar usando a Katana Cibernética.
2. **Evolução (Data-Nodes):** Coleta de dados hackeados para fortalecer status base.
3. **Desafio (Chefes):** Batalhas intensas contra os outros heróis corrompidos.
